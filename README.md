# RockPaperScissor

## Description
### This is a rock paper scissor game. This game also includes a bot to play with.

## How to install

### Clone the repo
```bash
git clone https://github.com/BaibhavPhaijoo/RockPaperScissor
```
### Go to the directory of the repo
```bash
cd RockPaperScissor
```

### Run the code
```bash
dotnet run
```
